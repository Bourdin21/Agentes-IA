# Configuración General de Agentes de IA — OlvidataSoft

Este directorio contiene configuraciones y directrices generales aplicables a cualquier agente de IA utilizado en los proyectos de OlvidataSoft.

## Archivos

- `agent-guidelines.md` — Directrices generales para el uso de agentes de IA.
- `.gitignore-ai` — Patrones de `.gitignore` recomendados para proyectos con IA.

## Buenas Prácticas

- Revisar siempre el código generado por IA antes de integrarlo.
- Mantener las claves de API fuera del código fuente.
- Documentar el uso de IA en cada proyecto.
