# GitHub Copilot — Configuración para Proyectos OlvidataSoft

Este directorio contiene plantillas y configuraciones para usar **GitHub Copilot** en los proyectos de OlvidataSoft.

## Archivos de Configuración

- `copilot-instructions.md` — Instrucciones personalizadas para el agente de Copilot (ubicado en `.github/`).
- `settings.json` — Configuración recomendada de VS Code para Copilot.

## Uso

1. Copiar `settings.json` en el directorio `.vscode/` del proyecto.
2. El archivo `.github/copilot-instructions.md` es detectado automáticamente por GitHub Copilot.

## Configuración Recomendada de VS Code

Ver `settings.json` para las opciones recomendadas.
