# OpenAI API — Configuración para Proyectos OlvidataSoft

Este directorio contiene plantillas y configuraciones para integrar la **API de OpenAI** en los proyectos de OlvidataSoft.

## Archivos de Configuración

- `agent-config.json` — Configuración base del agente (modelo, parámetros, etc.).
- `.env.example` — Variables de entorno necesarias (sin valores reales).

## Uso

1. Copiar `.env.example` a `.env` en la raíz del proyecto y completar con las claves reales.
2. Adaptar `agent-config.json` según las necesidades del proyecto.

> **Importante:** Nunca incluir claves de API reales en el repositorio. Usar variables de entorno.
